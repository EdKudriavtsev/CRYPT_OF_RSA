What it does

Немного о сервисе

Сервис поможет вам лучше разобраться в принципе работы основного алгоритма асимметричного шифрования, используемого по всему миру.

Шифрование текстов

Сервис позволяет шифрование и дешифрование текстов при помощи числовых ключей алгоритма RSA

Генерация ключей

Для полного понимания работы алгоритма сервис предоставляет возможность сгенерировать новые ключи для новых шифротекстов

Демонстрация

Сервис несет лишь ознакомительную функцию и не является программой для шифрования данных в полной мере.

Tips for users

Краткое пояснение работы алгоритма

Алгоритм шифрования
На данном сайте реализован основной алгоритм асимметричного шифрования(RSA), предполагающий наличие двух ключей(приватный и публичный) у каждого пользователя. Публичный ключ известен любому пользователю, а приватный держится в секрете.
Значение ключей
Как уже говорилось ранее, у каждого пользователя есть два ключа. Если пользователь А хочет отправить свое сообщение пользователю В, он использует публичный ключ пользователя В. Пользователь В же, в свою очередь, расшифровывает сообщение своим приватным ключом и получает исходный текст. Более подробное объяснение Вы найдете в разделе "Справка".
